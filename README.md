# FatelTale
Fatel Tale 是一 2D 彈幕射擊遊戲，在遊戲中，每個玩家將扮演一名角色，彼此協力闖關與敵人戰鬥並 設法擊敗 BOSS，同時競爭分數。

# Build

執行以下指令編譯：
* `make` : 編譯Client、Server
* `make client` : 編譯Client
* `make server` : 編譯Server
* `make clean` : 清理所有產生的檔案(*.class etc.)

# Run

執行Server：
```java
java Server
```

執行Client:
```
java Client SERVER_ADDR
```
